## 이 체크리스트는 매 태스크 완료 시 실행한다 (#6 Self-Review + #9 문서 + #12 재검증)

### 기능
- [ ] 변경한 코드가 의도대로 동작하는가?

### 테스트 (Evidence 필요)

- [ ] 빌드 통과 — 실행 결과 1줄 요약 (실패 시: 출력 첨부)
- [ ] 린트 통과 — 실행 결과 1줄 요약 (실패 시: 출력 첨부)
